# vifm
Vifm is a fast, lightweight and highly configurable file manager with vi-like key bindings.
## Features 
- Vi-style key-bindings
- Dual-pane file browsing
- Powerful file search and filtering
- Bookmarks and directory history
- Configurable file previews
- Extensive customization via config files
- Symbolic link support
- Archive browsing
- Works in terminal
- Fast and lightweight
## Installation 
Node.js apks ships with prebuilt vifm.

## Getting Started 
- Launch vifm through:
1. Type ```vifm``` in the terminal
2. Select vifm from the menu

## Configuration 
- Vifm is configured using ```~/.config/vifm/vifmrc```
## File Previews 
- Enable file previews in the config
## Symbolic Links 
- Create links
- Navigate links
- Copy links as links
## Scripting and custom commands 
- Define custom commands in vifmrc
## Supported File operations 
- Copy/move/rename
- Delete
- Permissions editing
- Bulk rename
- Archive handling
